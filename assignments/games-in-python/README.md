
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a Hangman game using Python to practice string handling, loops, conditionals, and user interaction.

## 📝 Tasks

### 🛠️ Build the game flow

#### Description
Create the main Hangman game loop and game structure for selecting a secret word, prompting for guesses, and updating progress.

#### Requirements
Completed program should:

- Use a predefined list of words.
- Randomly select one word for each game.
- Display the current word progress with underscores for unguessed letters.
- Accept letter guesses from the player.
- Reveal letters when they are guessed correctly.

### 🛠️ Manage guesses and game state

#### Description
Implement guess tracking, attempt limits, and game-ending conditions for winning or losing.

#### Requirements
Completed program should:

- Track letters already guessed and prevent duplicate entries.
- Count incorrect guesses and limit the number of wrong attempts.
- End the game when the word is guessed or when attempts run out.
- Display a win message if the player guesses the word and a lose message if they do not.
