# 📘 Assignment: Interactive Web Pages

## 🎯 Objective

Build a responsive interactive web page using HTML, CSS, and JavaScript that updates content based on user actions.

## 📝 Tasks

### 🛠️ Create the page structure

#### Description
Create a web page with a header, a main interaction section, and a footer. Add a styled form area where a user can enter their name and choose a theme.

#### Requirements
Completed page should:

- Include a structured HTML layout with header, main, and footer sections
- Add a form with a name input and a theme selector
- Use CSS to style the page with responsive spacing, colors, and hover effects


### 🛠️ Add interactive behavior

#### Description
Use JavaScript to respond to form input and button clicks. Update the page content without reloading the page.

#### Requirements
Completed page should:

- Display a greeting message after the user submits their name
- Update the page theme when the user selects a different style option
- Show at least one additional interactive element, such as a button that reveals a surprise message or toggles visibility
